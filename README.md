## 规则

### 命名风格

#### 驼峰命名法

名称命名有完整的单词组成，单词与单词间用首字母大写区分

##### 大驼峰命名法

变量名词首字母大写

```java
public class User{
    String FirstName = "三";
    String LastName = "张"; 
}
```

##### 小驼峰命名法

变量名词首字母小写

```java
String firstName = "三";
String lastName = "张";
```
